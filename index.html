Sobre la librería

Una librería local con catálogo cuidado y venta online inicial.

Infinito Libros acompaña a lectores, familias y estudiantes con recomendaciones personalizadas, productos seleccionados y una experiencia simple: buscás el libro, lo agregás al carrito y completás un pedido inicial.

Esta demo representa una web profesional para pequeños negocios, preparada para mostrar el flujo de compra. En producción, los pedidos y pagos requieren backend, base de datos e integración segura con Mercado Pago Checkout Pro.

Tienda online inicial

Elegí libros y armá tu pedido.

Esta demo muestra un carrito y checkout inicial. El stock se plantea como dato administrado manualmente por la librería. En producción, los pagos online se integrarían con Mercado Pago Checkout Pro y el pedido quedaría registrado en una base de datos.

Modo demo: importar catálogo desde Excel, CSV o JSON Actualiza la grilla en este navegador y permite descargar el JSON para publicar. +

Usá la plantilla incluida en el ZIP. Columnas esperadas: ISBN, Titulo, Autor, Categoria, Precio, Descripcion, Imagen, Destacado.

En un sitio estático, la importación desde el navegador no reemplaza automáticamente archivos del hosting. Para publicación real, se importa el Excel, se descarga el JSON actualizado y se sube como data/catalogo.json.

Modo demo: actualizar precios por ISBN Subí una planilla con ISBN y Precio para actualizar libros ya existentes en el catálogo. +

Esta herramienta simula el flujo mensual presupuestado: el archivo debe contener columnas ISBN y Precio. El sistema busca cada ISBN existente y actualiza el precio.

Si el ISBN no existe, no se crea un libro nuevo. Esos casos se informan como no encontrados para revisión.

Criterio operativo

Stock administrado manualmente

La tienda se plantea con control básico de stock actualizado por el equipo de la librería. Cuando se realice una venta en el local, el personal deberá ajustar el stock para mantener la web alineada con la disponibilidad real.

No se incluye sincronización automática con caja, ERP o ventas del local físico; esa mejora puede cotizarse en una etapa posterior.

Pagos

Mercado Pago Checkout Pro

El flujo de pago incluido se plantea mediante Mercado Pago Checkout Pro, redirigiendo al comprador al entorno seguro de Mercado Pago para abonar con débito, crédito u otros medios disponibles.

Webhooks avanzados, conciliación automática, promociones especiales, MODO/Payway u otras pasarelas quedan fuera del alcance inicial y pueden cotizarse como etapa posterior.
